# Autonomous Systems Lab
## Underwater Soft Robotic Eel

**Lab:** Autonomous Systems Lab  
**Advisor:** Prof. Mark Campbell  
**Institution:** Cornell University  
**Duration:** August 2025 - Present

## Project Overview

Designing and developing an underwater soft robotic eel capable of autonomous navigation. This project integrates mechanical design, control systems, and computer vision to create a bio-inspired underwater robot for exploration and research applications.

## Mechanical Design

### Head Module Design

**Pressure Vessel:**
- CNC-machined waterproof pressure vessel to house electronics
- Designed to withstand underwater pressure at operational depths
- Integrated waterproof cable connections to midsections

**3D-Printed Exoskeleton:**
- Custom-designed protective exterior structure
- Hydrodynamic shape for efficient underwater locomotion
- Modular design for easy assembly and maintenance

**Skills:** CNC Machining, 3D Printing, CAD Design

### Electronics Integration

**Components Housed:**
- **Raspberry Pi:** Central processing unit for control and autonomy
- **Camera System:** Vision for navigation and object detection
- **Battery System:** Power management for extended operation
- **Waterproof Connections:** Sealed cable pass-throughs to midsections

**Skills:** Raspberry Pi, Electronics, Waterproofing

## Control Systems

### Web-Based GUI

Developed a web interface for real-time robot control and monitoring:
- Live video streaming from onboard camera
- Real-time control interface for manual operation
- Telemetry display for robot status and sensor data
- Remote access capability for testing and demonstrations

**Skills:** Web Development, GUI, Live Streaming

### PID Controller Implementation

**Sensor Feedback:**
- **IMU (Inertial Measurement Unit):** Provides orientation and acceleration data
- **Bend Sensors:** Monitor body curvature for locomotion control

**Control Implementation:**
- Developed PID control algorithm for swimming locomotion
- Integrated sensor feedback for stable, efficient movement
- Tuned control parameters for optimal performance

**Skills:** PID Control, IMU, Sensor Fusion

## Computer Vision & Autonomy (In Progress)

Currently integrating computer vision capabilities for autonomous operation:
- **Target Detection:** Object recognition for navigation goals
- **Autonomous Navigation:** Path planning and obstacle avoidance
- **Vision-Based Control:** Using visual feedback for precise maneuvering
- **Mission Planning:** High-level autonomy for complex tasks

## Project Goals & Applications

This soft robotic eel demonstrates bio-inspired underwater robotics with potential applications in marine research, underwater inspection, and environmental monitoring. The combination of soft actuation, autonomous navigation, and computer vision creates a versatile platform for exploring underwater environments with minimal environmental impact.
